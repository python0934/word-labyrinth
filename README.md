# word-labyrinth
# 《字走迷城》游戏玩法说明

## 游戏目标
在9x9的迷宫中收集汉字部件，合成正确的汉字，击败敌人，突破障碍，最终到达终点的门并成功通关，同时获得尽可能高的得分。

## 基本操作
- **PC端**：使用方向键（↑↓←→）移动
- **移动端**：在屏幕上滑动来移动

## 核心玩法

### 1. 收集汉字部件
- 迷宫中的格子上有各种汉字偏旁部首
- 走到有部件的格子上会自动收集到收集袋中
- 收集部件可获得10分

### 2. 合成汉字
- 点击右侧的"合成"按钮
- 系统会检查收集袋中的部件，自动合成可组成的汉字
- 每合成一个汉字，生命值和攻击力各+1
- 合成汉字后，用于合成的部件会从收集袋中移除

### 3. 战斗系统
- 遇到敌人（错字兽）时会触发战斗
- **战斗胜利**：如果有可合成的汉字或攻击力>15，击败敌人并获得100分
- **战斗失败**：如果没有可合成的汉字且攻击力≤15，生命值-1并被击退回上一格

### 4. 障碍物系统
- 遇到障碍物时无法通过
- 当攻击力>10时，可以强制摧毁障碍物

### 5. 道具系统
- 迷宫中散布着各种道具，带有脉冲动画效果
- ⭐ 星星：攻击力+2
- 💎 宝石：+500分
- ❤️ 爱心：生命值+2
- ⚡ 闪电：提升移动速度
- 🧩 拼图：随机获得一个汉字部件
- 获得道具可获得50分

### 6. 通关条件
- 到达地图右下角的"门"字格子
- 此时需要有可合成的汉字才能打开门并通关

## 游戏状态
- **生命值**：初始3点，生命值为0时游戏结束
- **攻击力**：初始1点，影响战斗和摧毁障碍物的能力
- **得分**：通过收集部件、击败敌人和获得道具累积
- **收集袋**：存放收集到的汉字部件

## 游戏流程
1. 从左上角的起点开始
2. 在迷宫中探索，收集汉字部件
3. 合成汉字提升能力
4. 击败敌人和突破障碍
5. 收集道具获得额外效果
6. 最终到达终点的门并合成汉字通关

## 提示
- 合理规划路径，避开强敌和障碍物
- 优先收集道具提升能力
- 及时合成汉字以增强战斗力
- 注意生命值，避免不必要的战斗
- 尝试获得高分，挑战自己的极限
